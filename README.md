# contoh JWT dengan PHP Native

#Kebutuhan Awal
1. Composer
2. PHP
3. Web Server

## Instalasi
1. Install dependency
```bash
composer install
```
3. rename .env.example dengan nama .env, dan sesuaikan setting
4. Jalankan web server, arahkan ke port tertentu
```bash
php -S localhost:8000
```
